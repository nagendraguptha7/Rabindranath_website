# Rabindranath Tagore High School Website

A simple, responsive static landing page for **Rabindranath Tagore High School**. This project showcases a modern school website layout built using plain HTML, CSS, and JavaScript.

---

## 🚀 Features

- ✅ **Responsive layout** — works well on mobile and desktop
- ✅ **Hero section** with call-to-action buttons (WhatsApp admission links)
- ✅ **Sticky navbar** with smooth scrolling to page sections
- ✅ **Admissions announcement bar** at the top
- ✅ **Facilities section** with card layouts and icons
- ✅ **Academics section** describing grade-wise curriculum
- ✅ **Gallery** grid with hover overlays
- ✅ **Contact section** with map embed and contact info
- ✅ **Footer** with quick links and social icons

---

## 📁 Project Structure

```
index.html    # Main landing page
style.css     # Styles and responsive layout
script.js     # Navbar toggle + scroll behavior
README.md     # Project documentation (this file)
```

---

## 🛠️ How to run locally

Just open `index.html` in your browser.

No build step or web server is required for development — it is a pure static site.

---

## 🚀 Deployment

You can deploy this directly to any static hosting provider (Vercel, Netlify, GitHub Pages, etc.).

### Recommended (Vercel)
1. Push this repo to GitHub (already done).
2. Connect the repo in the Vercel dashboard.
3. Deploy — Vercel auto-detects it as a static site.

---

## ✨ Customization

- Update `index.html` to change text, images, and links.
- Update `style.css` to change branding colors and layout.
- Update phone or WhatsApp links to match your real contact numbers.
